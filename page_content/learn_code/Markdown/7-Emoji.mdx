# Emoji

Markdown also supports inclusion of emoji.  

The emoji can be initiated using the format  `:<emoji_code>:`

Some of these are:

Thumbs up      :+1:

Sparkles      :sparkles: 

Koala      :koala:

Tada!      :tada:

Rocket      :rocket: 

Metal      :metal: 

Octocat      :octocat:

Explore more of these awesome emoji codes at [Emoji Cheat Sheet](https://www.webfx.com/tools/emoji-cheat-sheet/)
